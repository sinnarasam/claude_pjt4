# 키보드 입력 테스트 (Keyboard Input Tester)

물리 키보드 입력 또는 화면 클릭 시 어떤 키가 눌렸는지 시각적으로 확인할 수 있는 웹 기반 키보드 테스트 도구입니다.

## 주요 기능

- **시각적 QWERTY 키보드** — 숫자열, 알파벳, 특수문자, Shift / Ctrl / Alt, 방향키까지 풀 사이즈 레이아웃
- **두 가지 입력 방식**
  - 물리 키보드 입력
  - 화면의 키 직접 클릭
- **실시간 하이라이트** — 키를 누르면 검정색으로 활성화
- **입력 디스플레이** — 입력한 텍스트가 상단에 누적 표시 (Backspace 동작 지원)
- **마지막 키 정보** — 가장 최근에 입력된 키 이름 표시
- **초기화 버튼** — 입력 내용 리셋
- **반응형 디자인** — 모바일 화면에서도 사용 가능

## 디자인

- 화이트 베이스의 미니멀 디자인
- 부드러운 보더와 미세한 그림자 처리
- 활성화된 키는 검정/흰색 반전으로 강조

## 실행 방법

브라우저로 바로 열기:

```bash
xdg-open index.html
```

또는 로컬 서버 실행:

```bash
python3 -m http.server 8000
```

그 후 브라우저에서 `http://localhost:8000` 접속.
